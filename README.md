# Sentinel-5P Smog Monitor: Islamabad

## Project Overview
This project uses Python and Google Earth Engine to analyze atmospheric Nitrogen Dioxide ($NO_2$) levels over the Institute of Space Technology (IST), Islamabad.

## Key Findings
* **Data Source:** Sentinel-5P TROPOMI Satellite.
* **Observation Period:** October - November 2025.
* **Result:** Detected a **300% increase** in $NO_2$ concentration during the onset of the smog season (Nov 1st - Nov 22nd).
